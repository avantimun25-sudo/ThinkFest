📝 Task & Notes Management System
📌 Overview

This is a web-based task management application built using JSP and Servlets. 
It allows users to organize daily tasks, manage lists, add tags, and track subtasks through a simple and user-friendly interface.

🚀 Features
Create, update, and delete tasks
Organize tasks (Today, Upcoming, Calendar)
Multiple lists (Personal, Work, etc.)
Tags for categorization
Subtasks support
Due date and task description
